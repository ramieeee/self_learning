# 2. 객체만들기

## 1) 클래스와 인스턴스

* 클래스: 객체를 만들 수 있는 틀 (클래스를 인스턴스를 만든다는 말)

```python
class User:  # class는 항상 대문자로 시작함
    pass  # pass는 아무 내용도 없다는 뜻

# 아래는 같은 클래스지만 다른 인스턴스들임
user1 = User()
user2 = User()
user3 = User()
```
